# IOHprofiler: IOHexperimenter

[This is](https://github.com/IOHprofiler/IOHexperimenter) the experimenter tool of the project __Iterative Optimization Heuristics Profiler__ (IOHprofiler). This tool provides a platform to test algorithms with generating standard data for the post-processing tool ([IOHanalyzer](https://github.com/IOHprofiler/IOHanalyzer)) and a benchmark set for pseudo Boolean problems. Importantly, this tool allows users to create new designed problems and customize format of output data.

Source code of this tool is built on C++, and API of C++ is supplied.

## Development environment
This code has been tested on gcc 5.4.0
